# Développement Multiplateforme Mobile — MAWANE (MII-BDCC, ENSET)

Travaux pratiques du module **Développement Mobile (Natif Android & Flutter)**,
Pr. **MAWANE**, Master MII-BDCC, ENSET — Université Hassan II Casablanca.

## Contenu

| Séance | Technologie | Description |

| Séance 1 | Android natif / Java | Calcul de l'IMC (catégorie + image). Mobile Natif. |

| Splash Screen, Navigation et Internationalisation | Android natif / Kotlin | Splash screen, navigation entre activités (Intent + Bundle), internationalisation FR/AR (Views classiques). |
